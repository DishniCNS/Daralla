# Studio Dashboard

React + TypeScript + Tailwind dashboard for Daralla internal studio projects.

## Environment

Copy `.env.example` to `.env`:

```bash
copy .env.example .env
```

Set:

- `VITE_API_BASE_URL` to the backend server
- `VITE_ADMIN_API_KEY` to match the backend admin key

## Run locally

```bash
npm install
npm run dev
```
